## Collect Onchain 1155 Tokens

The Protocol SDK can be used to prepare transactions for collecting 1155 tokens onchain. This is done using the `MintClient`.

### Setup/Create the MintClient

```ts twoslash
import { createMintClient } from "@zoralabs/protocol-sdk";
import { zora } from "viem/chains";

const mintClient = createMintClient({ chain: zora });
```

### Collecting an onchain 1155 token
